# Sistema Bancário em Python

Projeto desenvolvido em Python com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO) através da simulação de um sistema bancário simples executado no terminal.

## Funcionalidades

- Login de usuário
- Depósito em conta
- Saque de valores
- Consulta de saldo
- Cadastro de chave Pix
- Alteração de chave Pix
- Alteração de senha
- Validação de saldo para saques

## Conceitos Aplicados

- Classes e Objetos
- Encapsulamento
- Métodos e atributos
- Instanciação de objetos
- Estruturas condicionais
- Estruturas de repetição
- Organização de código orientada a objetos

## Estrutura do Projeto

### Classe `BankAccount`

Representa uma conta bancária contendo:

- Titular da conta
- Saldo
- Chave Pix
- Senha

### Métodos Implementados

- `deposit()`
- `withdraw()`
- `check_balance()`
- `has_pix()`
- `get_pix()`
- `set_pix()`
- `verification_pix()`
- `set_password()`

## Tecnologias Utilizadas

- Python 3

## Objetivo

Este projeto foi desenvolvido para praticar os fundamentos de Programação Orientada a Objetos (POO), simulando operações bancárias básicas e aplicando conceitos de modelagem de classes, atributos, métodos e interação com o usuário via terminal.
